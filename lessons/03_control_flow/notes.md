# Урок 03 — Control Flow

## Темы

| Конструкция | Описание |
|-------------|----------|
| `if / else if / else` | Условные ветвления |
| `switch / case` | Выбор по значению, `break`, fall-through |
| `for` | Цикл со счётчиком |
| `while` | Цикл с предусловием |
| `do-while` | Цикл с постусловием (минимум 1 итерация) |
| `break / continue` | Прерывание / пропуск итерации |
| range-based `for` | Итерация по коллекции (C++11) |

## Задание

Файл: `src/task01_control_flow.cpp`

**Угадай число** — программа загадывает число от 1 до 10, пользователь угадывает.  
После каждой попытки выводится подсказка: *Больше / Меньше / Угадал!*  
В конце выводится количество попыток.  
Бонус: если попыток > 5 — вывести "Следующий раз повезёт".

### Обязательно использовать
- `do-while` — основной цикл
- `if / else if / else` — подсказки
- переменную-счётчик попыток
