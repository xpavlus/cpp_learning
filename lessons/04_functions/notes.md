# Урок 04 — Functions (Функции)

## Объявление и определение функции

В Python функция создаётся через `def`. В C++ нужно указать **тип возвращаемого значения**, имя и типы параметров — всё строго статически.

```cpp
// Python: def add(a, b): return a + b
int add(int a, int b) {
    return a + b;
}
```

Порядок важен: функция должна быть **объявлена до** её вызова (либо через forward declaration).

```cpp
// forward declaration (прообъявление)
int multiply(int a, int b);

int main() {
    multiply(3, 4);  // OK, компилятор знает сигнатуру
}

int multiply(int a, int b) {
    return a * b;
}
```

## Значения параметров по умолчанию

Аналог `def greet(name="World")` в Python:

```cpp
void greet(std::string name = "World") {
    std::cout << "Hello, " << name << "!\n";
}

greet();          // Hello, World!
greet("Alice");   // Hello, Alice!
```

Параметры со значениями по умолчанию должны идти **в конце** списка параметров.

## Перегрузка функций (overloading)

В Python одна функция — одно имя. В C++ можно иметь несколько функций с одним именем, но **разными типами параметров**:

```cpp
int square(int x)      { return x * x; }
double square(double x) { return x * x; }

square(5);    // вызывает int-версию
square(2.5);  // вызывает double-версию
```

Компилятор выбирает нужную версию на этапе компиляции — это называется **статический полиморфизм**.

## inline-функции

Ключевое слово `inline` подсказывает компилятору «вставить тело функции прямо в место вызова» вместо обычного вызова через стек. Используется для маленьких часто вызываемых функций.

```cpp
inline int max(int a, int b) {
    return (a > b) ? a : b;
}
```

В современном C++ компилятор сам решает, инлайнить или нет — `inline` больше нужен для линковки (разрешает множественные определения в заголовочных файлах).

## Передача аргументов: по значению vs по ссылке

В Python объекты передаются «по ссылке» (точнее — по ссылке на объект). В C++ по умолчанию передача **по значению** (копия).

```cpp
void addOne_byValue(int x) { x += 1; }   // оригинал не меняется
void addOne_byRef(int& x)  { x += 1; }   // меняет оригинал

int n = 5;
addOne_byValue(n);  // n == 5
addOne_byRef(n);    // n == 6
```

`const`-ссылка используется, чтобы избежать копирования без права изменения:

```cpp
void print(const std::string& s) { std::cout << s; }
```
